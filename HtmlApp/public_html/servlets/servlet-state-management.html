1. Can post requests be bookmarked? What are the problems? 
	Post request can not be bookmarked because web browser bookmarks the url not the parameter in request body. 
To do this we can use the Postman extension for bookmarking and saving the request.

2. What is the purpose of request dispatching? 3. What is the difference between redirect and request dispatch?
	Request dispatching is the forwarding of the request to another JSP or servlets. 
In request dispatch the request and response will forward and web browser does not know about it. 
But in redirecting web browser get the status code of 30* so it will request to other redirected URL.

4. What is an attribute ? 5. What is the difference between attributes and parameters? 
	Attribute is an object on server used to promote the communication between components.
Attribute is name value pair which is string object which are read/write both and parameter are 
string objects only which is pass through form which are read only.

7. What does it mean to say that http is stateless? Give an example of a stateful protocol. 
	HTTP is stateless means it does not maintain any state of request after it been request
 and got response. It is handling each request independently.
	SMTP: Simple Mail Transfer Protocol is stateful protocol because it maintains the state.
 Each mail (transaction) goes from multiple request response stages.
